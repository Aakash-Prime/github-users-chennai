- This project scraped GitHub users from Chennai with over 50 followers.
- We gathered information about their public repositories.
- The analysis highlights the active development community in Chennai.

## Explanation of Data Scraping
I utilized the GitHub API to gather user and repository data. The process involved filtering users based on location and follower count, then fetching repository details for each user.

## Interesting Fact
A surprising number of users had diverse programming languages represented in their repositories, indicating a versatile tech community.

## Recommendation for Developers
Developers in Chennai should collaborate more on open-source projects to enhance visibility and engagement within the community.
